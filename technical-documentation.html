Web Development Fundamentals

A beginner-friendly reference guide covering HTML, CSS, and JavaScript — the three core technologies of the web.

This documentation was created as a vocational training project demonstrating full-stack web knowledge.

What is web development?

Web development is the process of building websites and web applications. It involves three core technologies working together:

HTML — defines the structure and content of a page.

CSS — controls the visual appearance and layout.

JavaScript — adds interactivity and dynamic behavior.

How they work together

<!-- HTML: structure -->
<button class="btn" id="myBtn">Click me</button>

/* CSS: style */
.btn { background: purple; color: white; }

// JavaScript: behavior
document.getElementById("myBtn").onclick = () => {
  alert("Hello!");
};
HTML

HTML structure

HTML (HyperText Markup Language) defines the structure and meaning of web content using elements called tags.

Basic document structure

Every HTML page starts with a standard boilerplate. The <head> contains metadata, while <body> holds visible content.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width">
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to my page.</p>
  </body>
</html>

Semantic elements

Semantic tags give meaning to content, improving accessibility and SEO.

<header>  — top navigation / logo area
<nav>     — navigation links
<main>    — primary page content
<section> — thematic grouping of content
<article> — self-contained content block
<aside>   — sidebar or related content
<footer>  — bottom information

Common tags

<h1> to <h6>  — headings (h1 = largest)
<p>           — paragraph
<a href="">   — hyperlink
<img src="">  — image
<ul> / <li>  — unordered list
<div>         — generic container
<span>        — inline container
CSS

CSS styling

CSS (Cascading Style Sheets) controls the visual presentation of HTML elements — colors, layout, fonts, spacing, and more.

Selectors and properties

/* Element selector */
h1 {
  font-size: 2rem;
  color: #333333;
}

/* Class selector */
.card {
  background: white;
  border-radius: 8px;
  padding: 1rem;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

/* ID selector */
#header {
  background: #7c6ff7;
  color: white;
}

Flexbox layout

.container {
  display: flex;
  justify-content: space-between; /* horizontal */
  align-items: center;           /* vertical */
  gap: 16px;
  flex-wrap: wrap;
}

Responsive design

Use media queries to adapt layouts for different screen sizes.

/* Mobile first approach */
.grid {
  display: grid;
  grid-template-columns: 1fr; /* 1 column on mobile */
}

/* Tablet and above */
@media (min-width: 768px) {
  .grid { grid-template-columns: repeat(3, 1fr); }
}
JavaScript

JavaScript fundamentals

JavaScript adds interactivity and dynamic behavior to web pages. It runs directly in the browser.

Variables

const name = "Jubby";       // cannot be reassigned
let age = 21;               // can be reassigned
var city = "Delhi";         // old way, avoid using

Data types

const text = "Hello";          // String
const number = 42;             // Number
const isTrue = true;           // Boolean
const skills = ["HTML", "CSS"]; // Array
const user = { name: "Coco", age: 21 }; // Object

Functions

// Regular function
function greet(name) {
  return `Hello, ${name}!`;
}

// Arrow function (modern)
const greet = (name) => `Hello, ${name}!`;

console.log(greet("World")); // Hello, World!

Conditionals and loops

// If statement
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

// Loop through array
skills.forEach(skill => {
  console.log(skill);
});
DOM

DOM manipulation

The Document Object Model (DOM) lets JavaScript read and change HTML elements on the page in real time.

Selecting elements

// Select by ID
const btn = document.getElementById("myBtn");

// Select by class (returns first match)
const card = document.querySelector(".card");

// Select all matching elements
const cards = document.querySelectorAll(".card");

Changing content and style

// Change text
btn.textContent = "New text";

// Change HTML inside element
card.innerHTML = "<strong>Updated!</strong>";

// Change CSS style
btn.style.backgroundColor = "purple";
btn.style.color = "white";

Event listeners

btn.addEventListener("click", () => {
  console.log("Button clicked!");
});

// Mouse hover
card.addEventListener("mouseover", () => {
  card.style.transform = "scale(1.05)";
});

Always add event listeners after the DOM is loaded. Place your <script> tag just before the closing </body> tag.
